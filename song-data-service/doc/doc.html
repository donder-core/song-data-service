/random

Fetch info about randomly selected songs.

Field Type Description
genre integer (Optional) The ID of a genre. Can be one of the following:
1 (Pop)
2 (Anime)
3 (Kids)
4 (Vocaloid)
5 (Game Music)
6 (Namco Original)
7 (Variety)
8 (Classical)
diff integer (Optional) The ID of a difficulty. Can be one of the following:
1 (Easy)
2 (Normal)
3 (Hard)
4 (Oni)
5 (Ura)
level integer (Optional) The level of a given chart. If diff is not specified, it will search all difficulties that have the given level. Must be between 1-10.
limit integer (Optional) Specify how many songs can be returned in a single request. Default value is 1.
include_sayonara boolean (Optional) Include songs which are no longer available in all countries. Default value is false.

Example:

/random?limit=5
Note: Your results may be limited, as the server can determine the maximum number of results to return.

/search

Search for song IDs with the given parameters.

Field Type Description
title string (Optional) Segment of a song's title. Searches all official titles. Also searches unofficial titles if use_alias is allowed.
subtitle string (Optional) Segment of a song's subtitle. Searches all official subtitles.
genre integer (Optional) The ID of a genre. Can be one of the following:
1 (Pop)
2 (Anime)
3 (Kids)
4 (Vocaloid)
5 (Game Music)
6 (Namco Original)
7 (Variety)
8 (Classical)
diff integer (Optional) The ID of a difficulty. Can be one of the following:
1 (Easy)
2 (Normal)
3 (Hard)
4 (Oni)
5 (Ura)
level integer (Optional) The level of a given chart. If diff is not specified, it will search all difficulties that have the given level. Must be between 1-10.
include_sayonara boolean (Optional) Include songs which are no longer available in all countries. Default value is false.
use_alias boolean (Optional) Use unofficial/alias titles when searching for songs via. title. Default value is true.
title_comparison string (Optional) Determine how to compare the results of title and subtitle. When set to and, only return the IDs that match both conditions. When set to or, return IDs that match either conditions. Default value is and.
limit integer (Optional) Specify a maximum amount of IDs to return. Default value is specified by the server.

Example:

/search?title=taiko&level=10

Response:

[425,593,879,1066,1072]
Note: Your results may be limited, as the server can determine the maximum number of results to return.

Values

/song or /random

Field Type Description
title_list [{string: string}] A list of official titles, with locales as keys and titles as values.
subtitle_list [{string: string}] A list of official subtitles, with locales as keys and titles as values.
alias_list [{string: string}] A list of unofficial alias used for searching, with locales as keys and titles as values.
genre integer The main genre as an ID. Possible values:
1 (Pop)
2 (Anime)
3 (Kids)
4 (Vocaloid)
5 (Game Music)
6 (Namco Original)
7 (Variety)
8 (Classical)
genre_list [integer] A list of all applicable genres as IDs.
region_list [{string: integer}] A list of the regions indicating availability status, with regions as keys and IDs as values. Possible values:
-1 (Unknown)
0 (Unavailable)
1 (Available)
2 (Campaign)
3 (Campaign - Wasn't active in this region)
4 (Don Medal Shop)
5 (AI Battle Unlock)
6 (QR Code)
7 (Transfer - Obtained via. playing this song with a second user who has already unlocked it)
chart_list [{integer: ...}] A list of playable charts. Possible keys:
1 (Easy)
2 (Normal)
3 (Hard)
4 (Oni)
5 (Ura)

chart_list values

Field Type Description
level integer The chart's difficulty level. Can be between 1-10.
style_list [{integer: [{string: integer}]}] A list of note counts for each style. Possible keys:
0 (Single Chart)
1 (SouUchi Chart - P1)
2 (SouUchi Chart - P2)
source_list [{string: string}] A list of websites where more information can be found about this specific difficulty, with website IDs as keys and URLs as values.

style_list values

Field Type Description
normal integer The normal branch's note count. If this chart is not branching, the note count will be listed here regardless.
expert integer The expert branch's note count. If this chart is not branching, this variable will not be listed.
master integer The master branch's note count. If this chart is not branching, this variable will not be listed.
